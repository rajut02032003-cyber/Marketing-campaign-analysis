
# 📊 Marketing Campaign Analysis

## 📌 Project Overview

This project focuses on analyzing customer data from multiple marketing campaigns to uncover insights about customer behavior, spending patterns, and campaign effectiveness.

The goal is to help stakeholders make data-driven decisions by identifying high-value customers and successful marketing strategies.

---

## 🛠️ Tools & Technologies Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL (Data storage and querying)
* Power BI (Dashboard & Visualization)
* Jupyter Notebook

---

## 📂 Project Structure

```
marketing-campaign-analysis/
│
├── data/
│   └── marketing_data.csv
│
├── notebook/
│   └── analysis.ipynb
│
├── README.md
```

---

## 🔍 Key Analysis Performed

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Customer Segmentation (Age, Income, Country)
* Spending Analysis across Product Categories
* Campaign Response Analysis (AcceptedCmp1–5, Response)

---

## 📈 Dashboard Features (Power BI)

* Customer Demographics Overview
* Spending Patterns by Product Category
* Campaign Performance Analysis
* Interactive Filters (Age Group, Country, Income)
* KPI Metrics (Total Customers, Total Spending, Response Rate)

---

## 💡 Key Insights

* High-income customers contribute the most to overall spending
* Certain product categories (like wine) dominate purchases
* Some campaigns show significantly higher response rates
* Customer segmentation helps target the right audience

---

## 🚀 How to Run the Project

1. Download the dataset from `/data` folder
2. Open the Jupyter Notebook (`analysis.ipynb`)
3. Run all cells step-by-step
4. Use the processed data for Power BI dashboard
